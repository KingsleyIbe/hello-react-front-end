# Getting Started

_To get a local copy up and running follow these simple steps._

1. Clone the repo
   ```sh
   git clone https://github.com/KingsleyIbe/hello-react-front-end.git
   ```
2. Goto project directory
   ```sh
   cd hello-react-front-end
   ```

3. Run app
   ```sh
   npm install
   ```
